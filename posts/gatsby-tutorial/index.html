Gatsby Blog Customizing

블로그를 운영하기 위해 어떤 플랫폼들이 있는지 알아보다가 많은 개발자들이 깃헙페이지를 이용해 블로그를 커스터마이징하는것을 꽤 많이 봤다. 언젠간 블로그를 운영할 생각이 있었지만 귀찮은 것들이 한 둘이 아니라 언젠간 하겠지 하는 마음으로 블로그 운영을 미루고있었다. 하지만 곧 학기가 시작하기도하고 새로운 마음으로 공부한 내용들을 기록하고 싶은 마음이 들었다. 그래서 결국 깃헙페이지를 이용해서 나만의 블로그를 만들기로 마음먹었다. 그래서 오늘은 겟츠비를 이용해 블로그를 만들었던 과정을 공유하려고 한다.

1. 겟츠비란?

겟츠비(Gatsby)란 정적페이지 생성기로 여러 자바스크립트,css 파일 등을 빌드해서 하나의 정적 html파일로 변환시켜주는 서비스다. 정적페이지와 SPA는 html파일로 빌드시켜주는것은 동일하지만 spa는 하나의 html파일을 생성하고 DOM을 실시간으로 조작해서 클라이언트 사이드 랜더링을 하는 방식으로 작동하지만, 정적페이지는 빌드할때 정적인 html파일 여러개를 생성하고 서버사이드 랜더링 방식으로 동작한다.

겟츠비는 mark down 방식으로 포스팅할 수 있고, SEO, PWA 같은 기능들을 사용할 수 있도록 해주는 여러 플러그인들을 지원한다. 특히 ‘gatsby-stater’를 이용하면 다양한 템플릿을 제공받을수 있기때문에 처음부터 하드코딩으로 블로그를 만들어야하는 생고생을 줄여줄 수 있다. gatsby-starter 다양한 템플릿이 마련되어있기때문에 고르는 재미도 있다.

2. 폴더구조

gatsby cli를 이용해 gatsby 명령어를 사용할 수 있다. gatsby new [폴더이름] [선택한 gatsby-starter url] 을 이용하면 선택한 템플릿을 다운받을 수 있다. 개발환경 세팅은 많은 블로그들이 다루고 있으므로 넘어가도록 하겠다. 나는 gatsby-stater-lumen 템플릿을 선택했다. 템플릿마다 폴더구조는 상이할 수 있다. 기본적인 커스텀은 config.js 파일에서 해결할 수있다.

  1. 게시글과 페이지는 content폴더에서 md형식으로 만들 수 있다.
  2. 이미지는 static/media 폴더에 저장하면 된다. 또한 프로필 사진은 photo.jpg란 이름으로 static폴더에 넣어주면 된다.
  3. gatby build 명령어를 입력하면 public폴더에 빌드한 파일들이 저장되게 된다.
  4. src폴더에는 컴포넌트와 템플릿등을 관리한다. 컴포넌트에는 게시글, 레이아웃, 사이드바 등 컴포넌트의 구조와 디자인을 관리할 수 있다. 디자인적인 커스텀은 src폴더에서 대부분 할 수 있다고 보면된다.
  5. gatsby-config.js 파일에서는 플러그인등을 관리할 수 있다. lumen을 사용하면 이미 여러 플러그인들이 설치된 것을 볼 수 있다.

3. 커스터마이징

나는 분류하고 구조화하는 것에 집착하는 스타일이다. 그래서 매뉴부분을 커스텀하는데 많은 시간을 투자했다. 내가 원하는 매뉴는 포스팅할때 카테고리를 설정하면 자동으로 programming의 하위 매뉴로 들어가게 되고 그 매뉴를 눌렀을 때 선택한 카테고리에 해당하는 게시물만 보여주고 싶었다. 그리고 만약 카테고리가 Essay면 programming 말고 Essay 매뉴에 들어가게 하고 싶었다. 하지만 생각보다 쉽지 않았다. 겟츠비 템플릿을 사용하려면 폴더구조에 익숙해져야 하기때문에 하루종일 폴더를 뒤지고 분석하는데 시간을 썼다. 그리고 겟츠비는 기본적으로 graphQL을 사용하기때문에 이것에도 익숙해져야했다.

일단 src/components/Sidebar/Menu/menu.js 로 이동한다.

  <li className={styles['menu__list-item']} key={item.path}>

config에서 설정한 menu들을 가져와서 map함수를 이용해 리스트를 생성하는 것을 알 수있다. 그래서 커스텀할 매뉴는 config에서 설정하지 않고 menu.js에서 직접 구현해야 겠다고 생각했다.

<li className={styles['menu__list-item']}>       
  Programming
  <ul >
    (카테고리 내용들을 넣고싶다)
  </ul>    
</li>

프로그래밍이란 매뉴안에 카테고리들을 넣을 생각이었다. 카테고리 데이터를 어디서 가져올까 고민하다가 template폴더에 categories-list-template.js폴더에서 카테고리 데이터들을 불러오는 것을 보고 여기서 아이디어를 얻었다.

import { useSiteMetadata, useCategoriesList } from '../hooks';
const { title, subtitle } = useSiteMetadata();
const categories = useCategoriesList();

이 부분을 통째로 menu.js에 가져와서 카테고리데이터를 얻었고 다시 map함수를 이용해 카테고리 리스트를 만들었다.

<li className={styles['menu__list-item']}>       
  Programming
  <ul >

      <li  key={category.fieldValue}>
        <Link 
          to={`/category/${kebabCase(category.fieldValue)}/`}

        >
          {category.fieldValue} ({category.totalCount})
        </Link>
      </li>
    ))}
  </ul>    
</li>
